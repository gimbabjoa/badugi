# 🎴 바둑이 온라인 멀티플레이어

## 📁 파일 구조
```
badugi-server/
├── server.js          # Node.js 서버 (WebSocket + HTTP)
├── public/
│   └── index.html     # 멀티플레이어 클라이언트
└── badugi-solo.html   # AI 대전용 (서버 없이 단독 실행)
```

## 🚀 실행 방법

### 1. 서버 시작
```bash
node server.js
```
> 외부 npm 패키지 불필요! Node.js 내장 모듈만 사용

### 2. 접속
- 로컬: http://localhost:3000
- 같은 Wi-Fi: http://[내IP]:3000

### 3. 외부 친구에게 공유 (다른 네트워크)
```bash
# 방법 A: ngrok (추천)
npx ngrok http 3000
# → https://xxxx.ngrok.io 링크를 친구에게 공유

# 방법 B: 포트포워딩
# 공유기 설정에서 3000번 포트 열기
```

## 🎮 플레이 방법

1. **방장**이 방 만들기 → 방 코드 생성
2. **친구**들이 방 코드 입력하고 참가
3. 방장이 **바인 금액** 선택 (1만/2만)
4. 게임 시작!

## ⚙️ 게임 규칙

- 4장의 카드로 서로 다른 무늬+숫자 조합을 만드는 게임
- 3번의 교환 기회 (아침🌅/점심☀️/저녁🌙)
- 교환 사이마다 베팅 라운드
- **골프 (A-2-3-4)**: 최강 패
- **바둑이**: 4장 모두 다른 무늬+숫자
- **베이스/투베이스/원카드**: 조건 충족하는 카드 수에 따라

## 🔧 포트 변경
```bash
PORT=8080 node server.js
```
